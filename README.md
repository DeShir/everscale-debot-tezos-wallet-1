# Tezos Wallet DeBot-1 

## Description
The project implements Tezos Wallet base on DeBot technology. 
For working this DeBot there is needs helper service(https://github.com/DeShir/everscale-helper) to be up.

## Usage
- Compile<br>
`./compile.sh`
- Deploy<br>
`./deploy.sh`
- Run
`./run.sh`
