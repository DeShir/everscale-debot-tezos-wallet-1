# Tezos Wallet DeBot-1 

## Description
The project implements Tezos Wallet base on DeBot technology. 
For working this DeBot there is needs to be up helper service https://everscale-helper.herokuapp.com/.

## Usage
- Compile<br>
`./compile.sh`
- Deploy<br>
`./deploy.sh`
- Run
`./run.sh`
